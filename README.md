# Transitland VS Code Extension

GTFS feed tools, DMFR editing, and Transitland integration for Visual Studio Code.

- DMFR language support with JSON schema validation, snippets, and formatting
- CodeLens on each feed showing live status from the Transitland archive
- Validate and inspect GTFS feeds directly from the editor
- Copilot tools and `@transitland` chat participant for DMFR editing assistance
- Standalone MCP server (`out/mcp-server.js`) for use with Claude and other AI tools

Requires the [transitland CLI](https://github.com/interline-io/transitland-lib) for feed validation and inspection.
